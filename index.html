_FlexBox_

Content in flexbox is laid in rows and colunms, based on the maker's preferences. They can be tiled or form different sized flex items(item boxes) that contain words, images or other things.

They resize to browser windows based on given parameters in the head of the html page

.flex-container > div {} needed to add into the head element so flexbox element is usable
display: flex; /* or inline-flex */ defines flex container
flex-direction: row | row-reverse | column | column-reverse;what direction the flex container is placed
flex-flow: column wrap;shorthand for flex direction and wrap together
flex-wrap: nowrap | wrap | wrap-reverse;wether flexboxes will try and fit onto a single line, or multiple
justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly | start | end | left | right ... + safe | unsafe;wether the flexboxes are aligned left, right or
align-items: stretch | flex-start | flex-end | center | baseline | first baseline | last baseline | start | end | self-start | self-end + ... safe | unsafe;how flexobjects align on Y axis
align-content: flex-start | flex-end | center | space-between | space-around | space-evenly | stretch | start | end | baseline | first baseline | last baseline + ... safe | unsafe;how flexobjects align on X axis
display: flex; { ... gap: 10px; gap: 10px 20px; /* row-gap column gap */ row-gap: 10px; column-gap: 20px;defines flexbox gaps }
.item {} needed to add into the head element so flexbox element is usable
order: 5; /* default is 0 */ what order the boxes appear
flex-grow: 4; /* default 0 */how large given box is
flex-shrink: 3; /* default 1 */how much an item can shrink
flex-basis: | auto; /* default auto */default size before item is distributed
flex: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ]shorthand for flex-grow, flex-shrink and flex-basis combined
align-self: auto | flex-start | flex-end | center | baseline | stretch;This allows the default alignment (or the one specified by align-items) to be overridden for individual flex items.
